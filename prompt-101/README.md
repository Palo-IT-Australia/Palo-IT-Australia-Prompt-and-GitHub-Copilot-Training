# Prompt 101

Welcome to **Prompt 101** — a practical introduction to prompt engineering for teams working in product discovery and software delivery.

## What you will learn

By the end of this module you will be able to:

- Explain what a prompt is and why quality matters
- Apply core prompting techniques (zero-shot, few-shot, chain-of-thought, role prompting)
- Write effective prompts for common product-discovery tasks
- Iterate on prompts to improve AI output quality

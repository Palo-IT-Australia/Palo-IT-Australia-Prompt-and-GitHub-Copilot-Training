# Prompt and GitHub Copilot Training

Training materials for product and engineering teams covering prompt engineering fundamentals and GitHub Copilot. Part of the PO with AI training module.

---

## Modules

### 🧠 [Prompt 101](prompt-101/README.md)

A practical introduction to prompt engineering for product teams.

| Topic | File |
|---|---|
| What is a prompt and why it matters | [01-introduction.md](prompt-101/01-introduction.md) |
| Core prompting techniques (zero-shot, few-shot, chain-of-thought, role prompting) | [02-prompt-techniques.md](prompt-101/02-prompt-techniques.md) |
| Hands-on exercises | [03-exercises.md](prompt-101/03-exercises.md) |
| Product discovery prompt library | [examples/product-discovery-prompts.md](prompt-101/examples/product-discovery-prompts.md) |

**Audience:** Product owners, business analysts, product managers, anyone using AI assistants  
**Prerequisites:** Access to any AI assistant (ChatGPT, Claude, Microsoft Copilot, etc.)  
**Time:** ~90 minutes

---

### 🤖 [GitHub Copilot 101](github-copilot-101/README.md)

A practical introduction to GitHub Copilot for developers and technical team members.

| Topic | File |
|---|---|
| What is GitHub Copilot and how it works | [01-introduction.md](github-copilot-101/01-introduction.md) |
| Key features: inline completions, Chat, CLI, PR integration | [02-features.md](github-copilot-101/02-features.md) |
| Hands-on coding exercises | [exercises/](github-copilot-101/exercises/README.md) |

**Audience:** Software engineers, technical leads, QA engineers  
**Prerequisites:** GitHub Copilot access, VS Code or JetBrains IDE  
**Time:** ~90 minutes

# GitHub Copilot 101

Welcome to **GitHub Copilot 101** — a practical introduction to GitHub Copilot for developers, product engineers, and technical team members.

## What you will learn

By the end of this module you will be able to:

- Explain what GitHub Copilot is and how it works
- Use Copilot's key features: inline completions, Copilot Chat, and CLI assistance
- Write effective prompts inside your code editor to get high-quality suggestions
- Apply Copilot responsibly and understand its limitations

## Prerequisites

- A GitHub account with GitHub Copilot access (Individual, Business, or Enterprise)
- VS Code, JetBrains, or another supported IDE with the Copilot extension installed
- Basic familiarity with at least one programming language

## Estimated time

~90 minutes (reading + exercises)

## Setup

1. Install the **GitHub Copilot** extension in your IDE
   - VS Code: [marketplace.visualstudio.com](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)
   - JetBrains: available via JetBrains Marketplace
2. Sign in with your GitHub account
3. Verify the Copilot icon appears in the IDE status bar

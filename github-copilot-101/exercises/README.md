# GitHub Copilot 101 — Exercises

These hands-on exercises are designed to be completed in your IDE with GitHub Copilot enabled.

## Exercise overview

| # | File | Skill practised | Language |
|---|------|----------------|----------|
| 1 | [starter/01_inline_completions.py](starter/01_inline_completions.py) | Inline completions from comments | Python |
| 2 | [starter/02_test_generation.js](starter/02_test_generation.js) | Generating unit tests | JavaScript |
| 3 | [starter/03_refactoring.ts](starter/03_refactoring.ts) | Refactoring with Chat | TypeScript |
| 4 | [starter/04_debugging.py](starter/04_debugging.py) | Debugging with Copilot | Python |
| 5 | [starter/05_documentation.js](starter/05_documentation.js) | Generating documentation | JavaScript |

## How to use these exercises

1. Open the relevant starter file in VS Code (or your preferred IDE)
2. Read the instructions at the top of each file
3. Use Copilot inline completions and/or Copilot Chat to complete the tasks
4. Experiment — try different prompts and compare results

## Tips

- If a suggestion is not what you want, press `Esc` to dismiss it, add more context to your comment, and wait for a new suggestion
- Use `Alt + ]` / `Option + ]` to cycle through alternative suggestions
- For Copilot Chat: select the relevant code before asking a question so Copilot has context
